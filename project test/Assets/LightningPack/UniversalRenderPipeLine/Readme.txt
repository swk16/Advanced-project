This package includes 3 Lightning effects.
(Mesh, Thunder and LightningMagic)

----------------------------
Quick Start
----------------------------
Open "Demo.unity"
(Assets/LightningPack_URP/Demo.unity)
hit Play button, You can see the particles!

----------------------------
Change the Color
----------------------------
By using the ParticleSystem/StartColor
You can change particle color.
